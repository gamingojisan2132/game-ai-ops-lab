# 次タスク実装用プロンプト

あなたはこのリポジトリの継続開発を担当するCodex / AIエージェントです。

`docs/TASKS.md` から次のタスクを1つだけ選び、小さく実装してください。

## 作業手順

1. `AGENTS.md` を読んで作業ルールを確認する。
2. `docs/TASKS.md` を読んで未完了タスクを確認する。
3. `docs/WORKLOG.md` を読んで直近の作業と未確認事項を確認する。
4. 今回着手するタスクを1つだけ選ぶ。
5. 作業範囲、完了条件、確認方法を短く宣言する。
6. 実装またはドキュメント更新を行う。
7. 必要な確認を実行する。
8. `docs/TASKS.md` と `docs/WORKLOG.md` を更新する。

## 制約

- 大きな変更を一度に行わないでください。
- 1タスク1目的を守ってください。
- 原因不明のまま依存関係を増やさないでください。
- エラーを隠すためにテストやビルドを無効化しないでください。
- フロントエンドを変更した場合は、実画面表示とブラウザコンソールエラーを確認してください。
- 実画面確認できない場合は「未確認」と記録してください。
- 回避策を使った場合は、理由、通常の直し方、デメリット、将来戻すべきか、環境問題かプロジェクト問題かを記録してください。
