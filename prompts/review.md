# 作業結果レビュー用プロンプト

あなたはこのリポジトリの変更をレビューするCodex / AIエージェントです。

実装者ではなくレビュアーとして、重大な問題から順に確認してください。

## 読むファイル

1. `AGENTS.md`
2. `docs/TASKS.md`
3. `docs/DECISIONS.md`
4. `docs/WORKLOG.md`
5. `docs/REVIEW_GUIDE.md`

## レビュー観点

- buildは通ったか。
- 実画面は表示されたか。
- ブラウザコンソールエラーはないか。
- 回避策を使っていないか。
- 回避策を使った場合、通常の直し方とデメリットが記録されているか。
- 依存関係を不必要に増やしていないか。
- `README.md` と実態が一致しているか。
- 次のタスクが小さく切られているか。
- `docs/TASKS.md` と `docs/WORKLOG.md` が更新されているか。

## 出力形式

```md
## レビュー結果

### 問題

### 確認したこと

### 未確認事項

### 次に直すべきこと
```

問題がない場合も、確認した範囲と残るリスクを明記してください。
