# 初期セットアップ用プロンプト

あなたはこのリポジトリで作業するCodex / AIエージェントです。

まず実装せず、リポジトリの状態を把握してください。

## 読むファイル

1. `AGENTS.md`
2. `README.md`
3. `docs/TASKS.md`
4. `docs/DECISIONS.md`
5. `docs/WORKLOG.md`
6. `docs/GAME_IDEAS.md`
7. `docs/REVIEW_GUIDE.md`

## 依頼

- このリポジトリの目的を短く要約してください。
- 未完了タスクを確認してください。
- 直近の作業ログから、前回どこまで進んだか確認してください。
- 仕様判断・技術判断で注意すべき点を確認してください。
- 次に着手すべき小さなタスクを1つだけ提案してください。

## 制約

- まだコード変更はしないでください。
- 人間が読む文章は日本語で書いてください。
- Windows環境を前提にしてください。
- Node / npm 系の作業では、ローカルでは `npm.cmd` を優先してください。
- 実画面確認できないものは「未確認」と明記してください。
